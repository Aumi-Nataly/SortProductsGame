# SortProductsGame
2DAndroid-игра.
Быстрая игра на логику: успевай за таймером! Тебе показывают картинку — определи, к какой из двух категорий она относится, или отправь в корзину, если не подходит ни к одной. Тренируй мышление, расширяй кругозор и ставь новые рекорды по очкам.

🛠️**Основные механики:**
* Управление реализовано через Unity **Input System**.
* Информация о текущем уровне хранится в **PlayerPrefs**.
* Информация о категориях уровня и товары хранится в json-файлах. Асинхронное чтение данных с через инструменты **UniTask**.
* Простая анимация появления товаров реализована через **DOTween**.
* Используется **Object Pooling** для продуктов, что позволяет переиспользовать объекты и снижать нагрузку на производительность.
* UI реализован через **Canvas**.
* Проект состоит из двух сцен: главное меню и сцена игры.

⚙️ **Техническая информация:**
